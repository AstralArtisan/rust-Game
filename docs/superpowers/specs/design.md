## 4. 强化系统（Augment）

### 4.1 结构

- 固定 3 层：基础 → 进阶 → 完美
- 第 3 层产生质变效果
- 4 类别：近战(Melee)、远程(Ranged)、机动(Mobility)、通用(General)
- 3 稀有度：普通、精英、传说

### 4.2 获取途径

- 战斗房通关：必定掉落 1 个强化选择（3 选 1）
- 精英房通关：掉落精英/传说级强化选择
- 奖励房"强化锻造"：升级已有强化
- 商店：购买强化或强化升级服务
- 事件房：特定事件可获得强化
- Boss 宝箱：精英/传说强化（3 选 1）

### 4.3 升级途径

- 再次获得同一强化时自动升级
- 奖励房"强化锻造"选择升级
- 商店购买"强化升级服务"（120 金币）

## 5. 能量与终结技系统

### 5.1 能量机制

- 上限默认 100，可通过升级/商店/强化提升
- 获取：近战命中 +2、远程命中 +1、击杀 +5、拾取能量球 +15
- 不自然回复，鼓励主动战斗

### 5.2 终结技槽位

- 4 个槽位，按数字键 1-4 释放
- 解锁：1 号初始，2 号第 2 层，3 号第 3 层，4 号第 4 层
- 获取：Boss 宝箱必给终结技选择，奖励房/商店也可获得
- 替换：获得新终结技时选择装入哪个槽位

## 8. 地图与房间流程

### 8.1 每层结构（10 房间）

```
房间 1：固定战斗房
房间 2-9：每次 2-3 个门，玩家选择
房间 10：固定 Boss 房
```

门生成规则：
- 必须包含至少 1 个战斗房选项
- 奖励房每层最多 1 次
- 商店/事件不连续出现
- 精英房从第 3 房间起可出现
- 只有一种合理选择时只生成 1 个门

每层预期：战斗 3-5 次、事件 1-2 次、商店 1-2 次、奖励 0-1 次、精英 0-1 次。

| 3 次 | "冒险者背包"：开局自带 1 治疗药水 + 1 能量药水 |
| 4 次 | "命运改写"：每层可免费重选一次门 |
| 5 次 | "大师起手"：开局从 3 个终结技中选 1 装备 |

难度选择：
- 普通：默认
- 困难（3 次通关解锁）：敌人 HP ×1.3，伤害 ×1.2，掉落 ×0.85
- 噩梦（5 次通关解锁）：敌人 HP ×1.6，伤害 ×1.4，掉落 ×0.7，精英必定双词缀，Boss 新增阶段

## 9. 多人模式

### 9.1 合作模式（Lightyear 0.26，主机权威）

模块结构：
- `coop/protocol.rs`：复制组件、消息、通道定义
- `coop/lobby.rs`：大厅连接、匹配、准备
- `coop/sync.rs`：实体/输入/状态同步
- `coop/authority.rs`：主机权威逻辑
- `coop/visuals.rs`：远程玩家视觉
- `coop/scaling.rs`：合作数值缩放
- `coop/ui.rs`：合作专用 UI

核心原则：
- 单机和合作共用同一套游戏逻辑，通过 `SessionMode` 资源区分
- 不在游戏逻辑中散布 if coop 分支，差异通过网络同步层处理
- 合作特殊逻辑封装在 coop 模块内

关键设计：
- 掉落物：两人独立拾取同一掉落物，各自获得
- 门选择：各自投票，相同直接进入，不同随机选一个
- 事件房：合作模式保留事件房
- 强化选择：各自独立
- 死亡：变幽灵态，通关当前房间后自动复活 50% HP
- 数值缩放：怪物 HP ×1.6，伤害 ×1.2，数量 +30%

### 9.2 PVP 模式（自写 UDP，轻量原型）

模块结构：
- `pvp/protocol.rs`：UDP 消息定义
- `pvp/net.rs`：网络收发
- `pvp/arena.rs`：竞技场逻辑
- `pvp/visuals.rs`：对手视觉
- `pvp/ui.rs`：PVP 专用 UI

原型功能：两人固定竞技场，近战+远程+冲刺对战，先死者输。预留扩展空间（道具、地图选择、多回合）。

## 10. UI 设计

### 10.1 整体风格

- 像素风 UI，与游戏美术统一
- 深色半透明背景面板
- 实现时通过 `/ui-ux-pro-max` skill 精细化设计
- 图标通过 `/game-assets` skill 生成像素风素材

### 10.2 主菜单（B 方案：左菜单 + 右场景）

左侧：
- 游戏标题 + Logo
- 菜单项：新游戏/继续游戏、查看存档、双人合作、双人对抗、成就、图鉴、设置、退出

右侧：
- 角色动画预览 / 动态场景

### 10.3 战斗 HUD

- 左上：HP 条 + 能量条
- 右上：楼层/房间信息、金币数
- 下方：终结技槽位（1-4），显示图标、CD、能量是否足够
- 右下：小地图（可选）

### 10.4 升级选择界面（三栏）

- 左栏：角色当前状态（HP、能量、属性、已有强化列表）
- 中栏：休息选项（篝火回血、冥想回能）
- 右栏：3 个强化选择（带图标、稀有度、效果描述）

升级时可从下列 7 个属性中随机抽 3 个，附加一个保底「回血」选项：

| 选项 | 增量 | 触底/上限 |
|------|------|----------|
| 攻击力 | +3 | – |
| 生命上限 | +15 | – |
| 移动速度 | +15 | – |
| 暴击率 | +5% | 80% 上限 |
| 近战攻击间隔 | -0.06s | 0.15s 下限 |
| 远程攻击间隔 | -0.04s | 0.15s 下限 |
| 冲刺冷却 | -0.10s | 0.30s 下限 |

数值来自 `assets/configs/rewards.ron::levelup`，便于策划/玩家直接调整。

### 10.5 暂停菜单

- 左侧：操作菜单（继续、存档、读档、成就、回主菜单）
- 右侧：RPG Maker 风格角色面板（头像、HP/能量条、属性、已装备强化标签）
- ESC 打开/关闭

### 10.6 Boss 宝箱界面

- 上排：终结技选择（2-3 选 1）
- 下排：强化选择（3 选 1）
- 底部：额外金币

### 10.7 其他界面

- 商店：三区并排（属性/强化/工具），底部刷新按钮
- 奖励房：三个服务卡片并排（疗愈/锻造/启示）
- 事件房：事件描述 + 多选项按钮
- 所有选择界面左侧或底部显示角色状态摘要
