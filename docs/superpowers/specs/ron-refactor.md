# 任务背景

在上文中，我们核查了游戏的数值设置，发现偏离了我们的“在ron文件中编写数值”的设定。此次任务，我们需要将数值统一至ron文件中，并让代码在实际运行时使用ron文件提供的数值。

这次改动只针对游戏已实现的功能，design中提到但未实现的功能先不纳入考虑范围。

# git

在仓库中，根据这次任务内容，创建新分支工作。

# 目标

对每个强化、事件、终结技等design中提到的部分进行逐一核实验证，保证功能正常、数值一致。

- 强化：在ron里似乎已经实现预期，但是仍需进一步的验证。确认无误后，以ron为准，修改该部分代码逻辑。
- 能量获取：先按ron来
- 终结技：按design来，写入ron，调整代码
- 事件：按design来，写入ron，调整代码
- 商店：检查rs中是否存在硬编码，并让ron与design同步
- 敌人数值需写入ron
- 精英：以design为准，统一入ron
- boss：按design修改
- 其他：先不做修改
